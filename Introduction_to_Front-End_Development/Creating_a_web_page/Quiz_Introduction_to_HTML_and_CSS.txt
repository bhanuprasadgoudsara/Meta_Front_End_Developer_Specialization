8. Based on the following CSS, what will be the margin-box width for div elements?

    padding-right: 5px;
    margin-left: 5px;

}
- 10 pixels
- 20 pixels
- 30 pixels - Correct
- 40 pixels
Ans:  The margin-box width = content width + padding width + border width + margin width. In this scenario, the margin-box width is 30 pixels.
